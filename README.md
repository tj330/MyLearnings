# MyLearnings  

This is a simple **Library** designed to help users manage and explore a curated list of programming books which helped me. The application features a **React** frontend for an intuitive user interface and a **Spring Boot** backend for robust API management and data handling.

---

## Features  

- Display a catalog of programming books with details such as title, author, publication year, and description.
- Detailed view of individual books.  
- Responsive and user-friendly design for seamless navigation.  

---

## Tech Stack  

### Frontend: React  
- **React** for building a responsive and dynamic user interface.  
- **Axios** for handling HTTP requests to the backend API.  
- **React Router** for implementing navigation.  
- **React Bootstrap** for styling (optional).  

### Backend: Spring Boot  
- **Spring Boot** for creating a RESTful API.  
- **Spring Data JPA** for database interactions.  
- **H2** database for storing book information.
